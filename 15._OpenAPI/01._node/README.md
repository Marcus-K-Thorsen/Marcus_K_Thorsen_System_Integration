# How to initial setup

```bash
$ npm init -y
$ npm i express
$ npm i swagger-jsdoc
$ npm i swagger-ui-express
```

Then add the following key-value pair in top-level of `package.json`:

```json
"type": "module"
```

Then you are ready to do the follwing in the `app.js` file:

# How to start application

```bash
$ nodemon app.js
```
